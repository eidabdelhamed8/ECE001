What is Operating Systems

is a computer system program designed to provide an operating platform for a group of other programs, and is responsible for managing parts, software, and resources of computer hardware. The intermediate operating system between the user and the computer is the operating system, which is the primary part responsible for operating the various user programs, and the use of operating systems is not limited to Computers only, so these systems are used in a different set of other smart devices such as mobile phones, video games, web servers, and supercomputers, and computer operating systems are specialized in carrying out the identification of input elements that are made through the keyboard, and sending outputs Various to the display screen to be displayed to the user.
